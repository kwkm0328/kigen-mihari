オンライン同期の設定

Firebase（無料）を保存先にして、PC・スマホで同じ期限を共有します。初回だけ下の設定が必要です（一度きり）。

Firebaseの準備手順（初回のみ・数分）
  1. console.firebase.google.com をGmailで開き「プロジェクトを作成」
  2. 左メニュー ビルド → Firestore Database →「データベースを作成」→ リージョンは asia-northeast1（東京）を選択
  3. プロジェクト設定（歯車）→「マイアプリ」→ </>（ウェブ）を追加 → 表示される firebaseConfig = { … } をコピー
  4. その中身を下の欄に貼り付け
  5. Firestore の「ルール」タブに、下の「推奨ルール」を貼って公開

推測されにくい長めの文字列にしてください。PC・スマホで同じ値を入れると同期します。

ONにすると、保存データを暗証番号で暗号化します（保存先の運営者にも中身は見えません）。他の端末でも同じPINが必要です。

暗証番号の入力

このデータは暗証番号で保護されています。表示するにはPINを入力してください。
